Readme.md
Este projeto ensina você a usar o Git.

git init - vai inicializar o projeto (criar o repositório)
git add Readme.md - o git add envia os arquivos para área do stage (próximo de entrar no palco)
git status - Changes to be committed (Alterações a serem confirmadas)
git commit -m "primeiro commit"
Para mudar a nomenclatura de 'master' para 'main'
git branch -M "main"
